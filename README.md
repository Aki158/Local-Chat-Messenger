# Local-Chat-Messenger
RecursionCSのBackendProject2_Local-Chat-Messengerのリポジトリーになります。

## 概要
UDP通信を用いてクライアント・サーバ間でのデータ送受信を行います。

- クライアント：ユーザーはコマンドライン上に入力しサーバに送信する。その後サーバから偽データを受信する。

- サーバ：クライアント側のデータを受信し、偽データを送信する。
